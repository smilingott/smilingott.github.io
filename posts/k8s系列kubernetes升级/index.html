K8S系列：Kubernetes升级

Posted by 文盲书生 on Fri, Apr 7, 2023
  • 查看并确定要升级到哪个版本
1yum list --showduplicates kubeadm --disableexcludes=kubernetes
  • 查看kubeadm版本
1kubeadm version
  • 升级第一个控制节点（一般为主控）
1yum install -y kubelet-1.25.0 kubectl-1.25.0 --disableexcludes=kubernetes
2kubeadm upgrade plan --ignore-preflight-errors=CoreDNSUnsupportedPlugins,CoreDNSMigration
3kubeadm upgrade apply v1.25.0 --ignore-preflight-errors=CoreDNSUnsupportedPlugins,CoreDNSMigration
  • 其他挖制节点
1kubectl drain host-X --ignore-daemonsets --delete-emptydir-data 
2kubeadm upgrade node --ignore-preflight-errors=CoreDNSUnsupportedPlugins,CoreDNSMigration --v=5
3yum install -y kubelet-1.25.0 kubectl-1.25.0 --disableexcludes=kubernetes
4systemctl daemon-reload
5systemctl restart kubelet
6kubectl uncordon host-X
  • 查看升级结果
1kubectl get pods -A | grep eted | awk '{print $2}' | xargs kubectl describe pod -n kube-system | grep Started
